컴파일 방법
1. Make 를 입력한다.
2. ./20141575.out 을 입력하면 프로그램이 실행된다.

실행방법
1. 프로그램을 실행한 후 shell 명령어를 입력한다.
2. shell명령어를 모를 경우 help 또는 h를 입력한다.
3. dir 또는 d를 입력하면 현재 폴더에 있는 폴더의 이름이 출력되고 디렉터리 파일은 뒤에 '/' 가 추가로 출력되고 실행파일 뒤에는 '*'가 출력된다.
4. quit 입력시 프로그램이 중지한다.
5. history 또는 h 입력시 지금까지 입력한 shell 명령어들을 출력해준다.
6. du 또는 dump를 입력시 160개의 메모리 내용이 출력되고 dump(du) start를 입력시 start 부터 160개의 메모리 내용이 출력된다.
   dump(du) start,end 를 입력시 start 부터 end까지의 메모리내용이 출력된다.
   형식에 맞지 않는 입력을 넣을 시에 오류를 출력한다.
   위의 명령어를 실행 후 dump를 입력시 출력된 메모리의 마지막 다음 주소부터 다시 메모리내용 160개 출력된다.
7. e address,value 를 입력시 address라는 주소에 value라는 값으로 편집된다.
   형식에 맞치 않는 입력을 넣을 시에 오류를 출력한다.
8. reset 을 입력시 메모리의 모든 내용이 0으로 바뀐다.
9. f(fill) start,end,value를 입력 시 start라는 주소에서 end라는 주소까지 value라는 값으로 채워진다.
   형식에 맞지 않는 입력을 넣을 시에 오류를 출력한다.
10. opcodelist 를 입력 시 58개의 opcode들이 hash table 형식으로 출력된다.
11. opcode mnemonic(명령어) 를 입력 시 그 명령어의 opcode를 찾아서 출력한다. 잘못된 명령어일 경우 오류가 출력된다.
12. 나머지 입력시 오류를 출력한다.
