5.사용법 
메뉴에서 3가지 경우의 수가 있는데
1을 누르면 로그인을 한후 게임을 실행할수 있고
2를 누르면 랭킹이 나오고
3을 누르면 프로그램이 종료된다.
게임방법은 행맨게임인데 character 힌트와 word pass라는 힌트가있다.
이아이템들은 1번씩만사용할수있다.

Manual 
There are 3cases in menu
When you press '1', you can login and play games
When you press '2', Print ranking
When you press '3', Program exit
This is Hangman game, you can use one character hint and word pass hint.
